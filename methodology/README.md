# 学习笔记方法论

## 核心理念

**笔记不是存档，是思考的副产品。**

好笔记不是"记录得多"，而是"思考得深"。工具只是执行者，方法才是核心。

---

## 第一阶段：捕获（学习时）

### 原则

| 原则 | 解释 |
|------|------|
| **最小摩擦** | 学习时不要打开笔记软件写长句，会打断思路 |
| **锚点优先** | 只记能触发回忆的关键词/时间戳 |
| **标记疑问** | 没听懂的地方立刻标记，不要骗自己 |
| **记录连接** | 想到和已有知识的关联立刻记下来 |

### 实际操作

学习中只记三样东西，每样不超过 10 个字：

```
⏱ 3:15 → Transformer 为什么不用 RNN
⏱ 8:40 → 多头注意力 [!]
🤔 softmax 的温度参数控制什么？
→ HMM（之前学过的）
```

格式：
- `⏱ 时间戳 + 关键词` — 方便回看
- `[!]` — 标记没听懂
- `🤔` — 自己的疑问/联想
- `→ 概念名` — 和已有知识的关联

## 第二阶段：整理（学完后 24 小时内）

### 核心规则

| 规则 | 为什么 |
|------|--------|
| **一张卡片只讲一个概念** | 方便复用、链接、复习 |
| **永远不复制粘贴** | 抄一万字不如写两百字，抄了就不是你的 |
| **必须用自己的话** | 即使不准确，也比原文有价值——因为暴露了你的理解盲区 |
| **必须关联至少一张已有卡片** | 孤立的知识记不住 |
| **24 小时内完成** | 超过 24 小时，记忆衰减到需要重看原素材 |

### 卡片格式

每张卡片五个部分：

```
## 概念名称

### 📺 原文引用
> ...（关键句，200 字以内）

### 📝 我的理解
用自己的话解释，300-500 字。
不追求一次准确——写你现在的理解，哪怕有遗漏。

### ❓ 知识盲区
写你还不确定的地方。这是复习时的重点目标。

### 🔗 关联概念
至少链接一张已有卡片。格式：[[卡片名]]
不知道有没有关联 → 那就是还没理解透。

### 💡 例子
一个生活化的例子，或一段可运行的代码。
```

### 对比自查

整理完后，检查每张卡片：

```
原文：Supervised learning learns a mapping function from X to Y
你的话：给模型例子让它学会预测

❌ 遗漏："映射函数"这个关键概念没提到
→ 说明你没完全理解，需要回看 3:15-5:00
```

关键不是"第一次就准确"。**发现遗漏 → 回看补上 → 彻底搞懂**，这本身就是内化的过程。

## 第三阶段：复习（每周）

### 复习方法

| 方法 | 频率 | 怎么做 |
|------|------|--------|
| **随机抽查** | 每周 1-2 次 | 闭眼随机抽 5 张卡片，尝试完整解释 |
| **填空自测** | 每次整理后 | 遮住答案，看标题复述 |
| **孤岛检查** | 每周 1 次 | 检查从未被引用的卡片：删掉，或强制链接到某个概念 |
| **MOC 更新** | 每月 1 次 | 重新审视知识图谱，调整分类和链接 |

### 卡壳处理流程

```
尝试复述卡片内容
    ↓
说不清？
    ↓
看卡片上的 [❓ 知识盲区]
    ↓
还是说不清 → 标记 #needs-review → 回看原素材对应片段
    ↓
说得清，但耗费时间太长
    ↓
说明理解不够深 → 简化卡片，去掉细节只留骨架
```

### 什么算"真正理解"

一张卡片你能做到三件事，才算真正内化：

1. **用自己的话对别人讲清楚**（费曼技巧）
2. **举出一个不一样的例子**（不是照抄卡片的例子）
3. **说出它和另一个概念的关联**（哪怕关联很弱）

## 第四阶段：分享

### 为什么必须分享

- 写给别人看，是最严格的"用自己的话"检验
- 别人提问 → 发现你的盲区 → 加深理解
- 知识不被使用，就是死的

### 分享方式

| 方式 | 适合 | 怎么做 |
|------|------|--------|
| 静态网站 | 完整的知识库 | ink 自动生成，部署 GitHub Pages |
| 单篇博客 | 单个话题 | 从卡片展开成文章 |
| 口头讲给别人 | 快速检验 | 找朋友花 5 分钟讲清楚一个概念 |

## 工具的角色

| 阶段 | 工具做什么 | 人做什么 |
|------|-----------|---------|
| 捕获 | 无 | 记锚点 |
| 整理 | LLM 拆分概念、生成卡片草稿、高亮遗漏 | 确认没歪、补充思考、关联卡片 |
| 复习 | 随机抽卡、填空自测、孤岛检测 | 复述、回看、调整 |
| 分享 | 自动生成网站 | 发布、回答问题 |

**工具不是帮你少思考，而是帮你把精力花在最重要的思考上。**

## 常见问题

**Q: 概念太复杂，200 字写不完怎么办？**
一张卡片 200 字写不完 → 说明这个概念还可以拆。拆成 2-3 张更小的卡片，每张一个子概念。

**Q: 找不到关联怎么办？**
找不到关联 = 还没理解透。说明这个概念是孤岛，要么你学得不深，要么它确实是个独立知识点。先标记出来，学深了自然能找到连接。

**Q: 总是忍不住复制粘贴怎么办？**
给自己一个"禁复制"规则：看到原文→关掉原文窗口→默写。实在写不出来，只写"关键句是 XXX，但我还不太懂"——比大段复制有用。

**Q: 感觉自己写得不好，不敢分享？**
写得不好 = 你的真实理解。分享它，别人指出问题 → 你就进步了。最糟糕的不是写得差，是把差的笔记锁在文件夹里自己都不看。
