📖 笔记方法论

记录阶段（看/读时）

只记时间戳 + 关键词 + 个人疑问，不写完整句子
没听懂的地方标记 [!] ，跳过回头补
有联想的立刻记 → [概念名]

整理阶段（24 小时内）

一张卡片只讲一个概念
永远不复制粘贴，用自己的话写
引用原文 → 自己的理解 → 知识盲区 → 例子
至少关联一张已有卡片

复习阶段（每周）

随机抽 5 张卡片，尝试复述
说不清的标记 #needs-review，回看原视频
检查孤岛卡片（无人引用），要么删要么强制链接
